# Better Notepad

Basically a clone of the steam notepad

## Features
- Autosave functionality
- Formatting buttons (B, I, U)
- Code block insertion with </>
- Basic calculator functionality (type equation then press or click =, ie "2+2=")

## Requirements
- Python 3.8+ 

## Run
```bash
python main.py
```
